## Projects

**[Hermes](https://github.com/DiegoGalloM/Hermes)**
AI-powered news pipeline: collects stories from NewsAPI and RSS, curates and summarizes them with the Claude API, and delivers a daily digest straight to Telegram.

**[CSV-Analyzer](https://github.com/DiegoGalloM/CSV-Analyzer)**
Interactive C++20 CSV analysis tool with adaptive type detection, template metaprogramming (`std::variant`), and statistical summaries rendered as ASCII histograms. Final project for Object-Oriented Programming.

**[Electromagnetic Particle Confinement](https://github.com/DiegoGalloM/Proyeto-Reto-Magn-tico)** · [live demo](https://magchallenge.netlify.app)
Simulated a charged particle confined by magnetic fields — from the Lorentz force to a 3D tokamak reactor. Implemented the RK4 integrator and the active, sectorized control system that keeps the particle confined in real time.
